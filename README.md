# SecurityCheckApp

Консольное приложение на C# для курсовой работы по дисциплине
«Методы и средства защиты информации в компьютерных сетях».

## Реализованные модули

1. Проверка подключения к Интернету (через `Ping`).
2. Проверка наличия межсетевого экрана и антивируса.
3. Проверка работоспособности межсетевого экрана.
4. Проверка работоспособности антивирусного ПО.
5. Вывод итогового отчёта.

## Запуск

```bash
dotnet run
```

> Приложение ориентировано на Windows (проверки МЭ/АВ используют `sc`, `netsh`, `powershell`, `tasklist`).
> На Linux/macOS выполняется только базовая часть (проверка сети), остальные проверки сообщат,
> что функциональность ограничена платформой.
