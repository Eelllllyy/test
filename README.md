# SecurityCheckApp

Проект теперь содержит **2 варианта запуска**:

1. **Windows UI (WinForms)** — графический интерфейс.
2. **Console (кроссплатформенный)** — для macOS/Linux/Windows.

## Важно

Вы запускали команду `dotenv run` — это другая утилита (для env-файлов).
Нужна команда **`dotnet run`**.

## Запуск UI (только Windows)

```bash
dotnet run --project SecurityCheckApp.csproj
```

## Запуск на macOS/Linux (консольный режим)

```bash
dotnet run --project SecurityCheckApp.Console.csproj
```

## Что проверяет приложение

1. Подключение к Интернету (`Ping`).
2. Наличие межсетевого экрана и антивируса.
3. Работоспособность межсетевого экрана.
4. Работоспособность антивирусного ПО.
5. Вывод итоговых результатов.

> Проверки МЭ/АВ через `sc`, `netsh`, `powershell`, `tasklist` полноценно работают только в Windows.
> В консольном режиме на macOS/Linux выводится корректное сообщение об ограничениях платформы.
