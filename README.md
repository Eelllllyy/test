# SecurityCheckApp

Оконное (WinForms) приложение на C# для курсовой работы по дисциплине
«Методы и средства защиты информации в компьютерных сетях».

## Что умеет приложение

1. Проверка подключения к Интернету (через `Ping`).
2. Проверка наличия межсетевого экрана и антивируса.
3. Проверка работоспособности межсетевого экрана.
4. Проверка работоспособности антивирусного ПО.
5. Вывод итогового отчёта в отдельное текстовое поле.
6. Очистка результатов и выход из программы.

## Как запустить (Windows)

1. Установить **.NET SDK 8.0**.
2. Открыть терминал в папке проекта.
3. Выполнить:

```bash
dotnet restore
dotnet run
```

## Интерфейс

После запуска откроется форма с кнопками:
- «Проверка подключения к интернету»
- «Проверка наличия МЭ и АВ»
- «Проверка работоспособности МЭ»
- «Проверка работоспособности АВ»
- «Вывод результатов»
- «Очистить»
- «Выход»

> Проверки МЭ/АВ используют системные команды Windows (`sc`, `netsh`, `powershell`, `tasklist`).
