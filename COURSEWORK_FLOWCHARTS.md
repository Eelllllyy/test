# Блок-схемы модулей курсовой работы

Ниже приведены простые блок-схемы по каждому модулю.

## 1.2. Разработка модуля проверки наличия подключения к Интернету

```mermaid
flowchart TD
    A([Начало]) --> B[Отправка ping на yandex.ru]
    B --> C{Ответ пришел?}
    C -- Да --> D[Передача в модуль вывода: Подключение есть]
    C -- Нет --> E[Передача в модуль вывода: Подключение отсутствует]
    D --> F([Конец])
    E --> F
```

## 1.7. Разработка модуля вывода результатов

```mermaid
flowchart TD
    A([Начало]) --> B[Чтение результатов всех модулей]
    B --> C[Формирование итогового текста]
    C --> D[Вывод в текстовое поле формы]
    D --> E{Нажата кнопка Сохранить?}
    E -- Да --> F[Сохранение отчета в txt-файл]
    E -- Нет --> G([Конец])
    F --> G
```

## Примечание

В курсовой работе модули межсетевого экрана и антивируса разделены:
- отдельный модуль проверки МЭ;
- отдельный модуль проверки АВ.
